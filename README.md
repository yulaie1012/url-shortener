# URL Shortener
A simple tool that takes a long URL and turns it into a short URL.

## Features
- create a short URL with 5 random numbers or letters

## Installation
1. Clone the repository
``$ git clone https://github.com/yulaie1012/url-shortener.git``
2. Install packages
``$ npm install``
3. Run seeder
``$ npm run seed``
4. Run app
``$ npm run dev``
